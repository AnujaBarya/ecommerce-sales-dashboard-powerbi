# Ecommerce Sales Dashboard

## Project Overview
This project showcases an interactive Ecommerce Sales Dashboard built using Power BI.  
The dashboard provides insights into sales performance, customer behavior, product categories, payment methods, and profit analysis.

---

## Tools Used
- Power BI
- Excel
- Data Visualization
- Dashboard Design

---

## Dashboard Features
- Total Sales & Profit Analysis
- Quantity and AOV Tracking
- State-wise Profit Analysis
- Category-wise Quantity Distribution
- Customer-wise Sales Analysis
- Payment Mode Analysis
- Monthly Profit Trends
- Quarter-wise Performance Filtering

---

## Key Insights
- Total sales amount reached 118K.
- Clothing category contributed the highest quantity share.
- COD was the most preferred payment method.
- Maharashtra generated the highest monthly profit.
- November recorded the highest profit among all months.
